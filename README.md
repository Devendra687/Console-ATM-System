# Console-ATM-System
Console-based ATM Management System in C with deposit, withdraw, and minimum balance validation.

ATM MANAGEMENT SYSTEM (C)

Overview
This is a console-based ATM Management System written in C.
The program simulates basic banking operations such as balance checking, deposit, and withdrawal with proper validation and minimum balance enforcement.

Features

• Check Account Balance

• Deposit Money

• Withdraw Money

• Minimum Balance Enforcement (₹1000 must be maintained)

• Insufficient Balance Handling

• Invalid Amount Validation

• Menu-Driven Interface

Tech Stack

Language: C

Library Used: stdio.h

How It Works

1. User selects an option from the menu.

2. Deposit increases the account balance.

3. Withdraw checks:

• Valid amount

• Sufficient balance

• Minimum balance condition

4. Program runs until the user exits.

How to Compile and Run

Compile: gcc atm.c -o atm

Run:./atm

(Windows users can run: atm.exe)
